REFUGE 2017

This is the ﬁrst project of my architecture studies, so the focus of this project is on learning the bases. The program required designing a refuge for an artist on a given site. The height of the ceiling, the orientation, the arrangement of the views and the general form of the project were designed to help the artist in the concentration process and allow for a certain calming atmosphere. A further focus was placed on the integration of the sloped site into the architectural project.
